# SoumyaATM-Simulator
My java project -
<br>
A Java-based ATM simulation project.
<br>
This is a GUI-based project using Java Swing, allowing users to perform basic ATM operations.
<br>
 # Features
- Secure Login
- Deposit & Withdraw
- Fast Cash
- Mini Statement
- PIN Change
- Transaction History Panel
- Daily Limit Check
  <br>
  <br>
# Technologies Used
- Java
- Java Swing (GUI)
- NetBeans or IntelliJ
- Ant Build System
